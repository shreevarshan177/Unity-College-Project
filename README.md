# Augmented Reality Drill Jig Visualization

## Overview
This academic project demonstrates the functionality of a drill jig using Augmented Reality (AR). A 3D drill jig model was designed in SolidWorks, integrated into Unity, and visualized in the real world using Vuforia image recognition.

## Objective
To develop an AR mobile application that allows users to scan a predefined image target and visualize a 3D drill jig model for interactive learning and demonstration.

## Technologies Used
- Unity
- Vuforia Engine
- C#
- SolidWorks

## Project Workflow
1. Designed the drill jig model in SolidWorks.
2. Imported the 3D model into Unity.
3. Configured a Vuforia image target for AR tracking.
4. Linked the 3D model to the image target.
5. Developed the Android application using C#.
6. Scanned the image target using the mobile application to display the drill jig in Augmented Reality.
